VIM highlighting for GPFS (Spectrum Scale)
==========================================

Author: Jez Tucker <jtucker@pixitmedia.com>


Installation
------------
Copy the ftdetect and syntax directories into your ~/.vim or your system vim directories.

Edit a .pol file.
